3. TAM to be added
4. Competitor offerings need manual work, have poor ratings for fucntionality, payment, and in-app ads.

Key findings:
1. Casual cricket bowlers are looking to measure their bowling speed and improve it, thus validating the need.
2. They understand speed guns are costly, and thus most of them have never measured it.
3. Majority of them aren't aware of such offering via mobile apps - Awareness Opportunity
4. Those who used it a few apps, valdiated that functionality is broken, manual, and inaccurate.
